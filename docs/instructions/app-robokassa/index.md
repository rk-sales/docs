# Robokassa APP — мобильное приложение

**Robokassa** — официальное мобильное приложение для быстрого управления магазинами, платежами и финансами прямо с телефона. Подходит для руководителей и продавцов.

## Что умеет приложение

- ✅ Просмотр платежей и возвратов в реальном времени
- ✅ Выставление счетов-ссылок на оплату
- ✅ Уведомления о важных событиях
- ✅ Работа с 2FA и биометрией

**Доступно для Android и iOS** — скачайте и войдите под своими данными из ЛК.

## Скачать приложение

| Платформа | Ссылка на установку |
|-----------|---------------------|
| **Android** | [Rustore](https://www.rustore.ru/catalog/app/ru.robokassa.pwa.twa){ target="_blank" }|
| **iOS** | [Web-приложение для Safari](https://pwa.robokassa.ru){ target="_blank" }|

Мобильное приложение для iOS доступно как **веб-приложение (PWA)**:

1. Откройте **Safari** на iPhone/iPad
2. Перейдите на **[pwa.robokassa.ru](https://pwa.robokassa.ru){ target="_blank" }**
3. Нажмите кнопку **«Поделиться»** (квадрат со стрелкой вверх)
4. Выберите **«На экран «Домой»»**
5. Подтвердите — иконка приложения появится на главном экране!
