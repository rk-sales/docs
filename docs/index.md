# О нас {: .about-title }

**Robokassa** — один из ведущих fintech‑сервисов для приёма онлайн‑платежей в России.  
Мы обеспечиваем защиту данных по стандарту PCI DSS и гарантируем стабильность платежей до 99,9% благодаря технологии [мультибанкинга](priem-platezhey/funktsii/multibanking.md).
Уже более 20 лет, мы помогаем запускать и развивать e‑commerce для ИП, юридических лиц и самозанятых.
