## Фискализация

Любой бизнес, включая самозанятых, обязан формировать и отправлять чеки покупателю и в ФНС в соответствии с 54‑ФЗ и 422‑ФЗ.  
Robokassa предлагает готовые решения, интегрированные с приёмом платежей, для любых форматов ведения бизнеса:

- **Robokassa Online** — облачная касса с интернет‑эквайрингом в одном договоре, поддержка 54‑ФЗ, ФФД 1.2, маркировки и любых систем налогообложения юрлиц;
- **Робочеки СМЗ** — автоматическая отправка чеков для самозанятых и ИП на НПД по 422‑ФЗ;
- **Робочеки** — сервис для юрлиц с небольшим объёмом чеков через облачную кассу Robokassa по 54‑ФЗ.
