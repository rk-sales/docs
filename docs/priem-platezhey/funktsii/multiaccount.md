# Мультиаккаунт

Мультиаккаунт в Robokassa — это возможность работать с **несколькими личными кабинетами разных компаний из одного аккаунта пользователя**.  
Вы входите по своему номеру телефона и дальше переключаетесь между ИП, ООО или самозанятыми без повторных логинов и паролей.

## Как устроен мультиаккаунт

- У пользователя есть **один аккаунт** на номер телефона.
- К этому аккаунту можно привязать **несколько ЛК разных юридических лиц**:
  - ИП,
  - ООО,
  - самозанятый (если предусмотрено),
  - магазины в разных юрисдикциях.
- Внутри интерфейса Robokassa вы просто выбираете, с каким кабинетом сейчас работать, — без выхода и нового входа.

Фактически это «единый вход» для владельца нескольких бизнесов или проектного менеджера.

## Зачем нужен мультиаккаунт

- **Один логин для всех компаний.** Не нужно держать десятки паролей, переключаться между браузерами или профилями.
- **Быстрое переключение.** Можно за пару кликов перейти из ЛК одного ИП в ЛК другого, посмотреть статистику или выставить счёт.
- **Удобство для агентов и владельцев групп компаний.** Если вы ведёте несколько проектов или брендов, ими проще управлять из одного аккаунта.
- **Меньше ошибок с доступами.** Не нужно передавать коллегам «общие логины» от разных кабинетов — каждому можно настроить свой доступ (через раздел «Сотрудники и доступ»).

## Пример использования

- У предпринимателя есть ИП для онлайн‑курсов и ООО для интернет‑магазина.  
  Он регистрирует один аккаунт по номеру телефона и внутри него подключает два ЛК — для ИП и для ООО.  
  Теперь он может:
  - смотреть отчётность по каждому бизнесу,
  - настраивать методы оплаты отдельно,
  - переключаться между кабинетами в пару кликов, не выходя из системы.

## Как работать с мультиаккаунтом

1. Зарегистрируйте аккаунт в Robokassa по номеру телефона.
2. Подключите первый ЛК (компанию) — заполните данные и пройдите стандартную активацию.
3. Для второй и следующих компаний добавьте новые ЛК через интерфейс: инструкция.
4. Используйте переключатель компаний в верхней части ЛК (или в меню), чтобы быстро переходить между кабинетами.
