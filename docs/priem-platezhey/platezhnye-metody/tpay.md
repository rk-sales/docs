# T‑Pay (Т‑Банк)

T‑Pay — способ оплаты для клиентов Т‑Банк без ввода данных карты.  
Клиент подтверждает платёж в приложении Т‑Банк, а вы получаете удобный и быстрый способ приёма оплат наряду с СБП.

## Ключевые преимущества

- **Без ввода реквизитов карты.** Все данные уже есть в приложении Т‑Банк, это снижает количество ошибок и повышает конверсию на этапе оплаты.
- **Меньше шагов до покупки.** Оплата занимает несколько секунд — клиенту достаточно подтвердить операцию в приложении.
- **Безопасность.** Данные карты не передаются магазину: используется токенизация и защита Т‑Банк, подтверждение через приложение и дополнительные факторы.

## Подключение через Robokassa

T‑Pay доступен после регистрации в Robokassa и подходит для ИП, ООО и самозанятых.  
Метод подключается вместе с другими платёжными решениями и может быть включён или отключён в настройках проекта при необходимости.

## Как платит клиент

1. Клиент выбирает способ оплаты T‑Pay на вашей платёжной странице.
2. Его перенаправляет в приложение Т‑Банк.
3. Клиент проверяет сумму и подтверждает операцию (пуш, SMS, биометрия).
4. Статус оплаты автоматически возвращается в ваш магазин, заказ считается оплаченным.
