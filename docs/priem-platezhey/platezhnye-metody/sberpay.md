# SberPay

SberPay — это способ оплаты для клиентов Сбера без ввода данных карты.  
Покупатель подтверждает платёж в приложении «СберБанк Онлайн», а вы получаете быстрые и безопасные онлайн‑платежи.

## Ключевые преимущества

- **Привычно клиентам Сбера.** Покупатель видит знакомый бренд своего банка и чаще завершает оплату.
- **Без ввода номера карты.** Все данные уже сохранены в приложении банка, меньше ошибок и недооплат.
- **Быстрая обработка.** Платёж подтверждается в несколько касаний, деньги зачисляются после успешной операции
- **Безопасность.** Подтверждение через приложение банка, биометрию или PIN снижает риск мошенничества.

## Подключение через Robokassa

SberPay подключается вместе с другими способами оплаты после регистрации в Robokassa.  
Достаточно добавить метод в проекте и включить его в настройках приёма платежей; отправка чеков входит в тариф эквайринга.

## Как платит клиент

1. Клиент оформляет заказ и выбирает способ оплаты SberPay.
2. Его перенаправляет в приложение «СберБанк Онлайн».
3. Клиент проверяет сумму и подтверждает оплату.
4. Статус заказа обновляется автоматически, оплата фиксируется сразу.
