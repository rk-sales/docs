# ЮMoney кошелёк

ЮMoney — электронный кошелёк, из которого покупатель может платить напрямую или через привязанную банковскую карту.  
Оплата проходит онлайн за пару шагов, а магазин получает деньги так же, как при обычной оплате картой.

## Как платит клиент

1. Покупатель оформляет заказ и выбирает способ оплаты «ЮMoney кошелёк».
2. Его перенаправляет на страницу ЮMoney, где при необходимости он входит в кошелёк.
3. Клиент выбирает, откуда списать деньги: с баланса кошелька или с привязанной к нему карты.
4. Подтверждает платёж паролем или одноразовым кодом. После этого возвращается на сайт магазина с готовым статусом оплаты.

## Преимущества для покупателя

- **Один кошелёк — много способов оплаты.** Можно платить с баланса кошелька или любой привязанной карты.
- **Быстрая оплата.** Постоянным пользователям достаточно пары кликов, без ввода длинных реквизитов.
- **Удобный учёт расходов.** Все платежи видны в истории ЮMoney, есть готовые шаблоны и сохранённые получатели.

## Преимущества для магазина

- **Дополнительная аудитория.** Вы получаете доступ к пользователям, которые предпочитают платить именно из кошелька.
- **Привычный сценарий оплаты.** Клиент остаётся в знакомом интерфейсе ЮMoney и с большей вероятностью заканчивает покупку.
- **Надёжные платежи.** Подтверждение операций через ЮMoney снижает число ошибок и незавершённых платежей.
- **Для покупателей из зарубежа.** Покупатели, которые по какой-то причине не могут оплатить своей иностранной картой, могут воспользоваться данным сервисом для совершения платежа.

## Подключение через Robokassa

Оплата из кошелька ЮMoney подключается в Robokassa в течение 14 рабочих дней.
